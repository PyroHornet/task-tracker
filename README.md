
# Task Tracker
A clean, responsive full-stack task management app for tracking daily to-dos.

## Live Demo
https://task-tracker-eight-green.vercel.app/

## Features
- Add, edit, mark complete, and delete tasks
- Mobile-friendly design
- Persistent storage (SQLite in progress)

## Tech Stack
- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: Node.js + Express REST API

## Local Setup
```bash
git clone https://github.com/SamaelGamboa/task-tracker.git
cd task-tracker
npm install

#Run frontend
npm run dev

#Run backend (separate terminal)
npm run server
```
